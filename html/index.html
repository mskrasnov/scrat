Scrat - консольный клиент Mastodon

Scrat - простой консольный клиент Mastodon с минимумом функций (отправка и чтение своих статусов; в будущем могут добавиться и другие функции). Он не поддерживает картинки, видео, радужные эмодзи и WEB-дерьмо (JavaScript и прочее).

Вы можете скачать Scrat здесь.

Стек технологий

  • Язык программирования: Rust
  • Операционная система: Windows, Linux
  • Интерфейс: в планах добавить псевдографический интерфейс посредством ratatui

RoadMap

Scrat всё ещё находится в разработке; до выхода версии 0.1 ещё далеко. Вот примерный план работ для этой версии:

  • Использовать ratatui для построения пользовательского интерфейса
  • Добавить возможность открытия профиля человека, который был упомянут в статусе с помощью @username
  • Добавить нормальную регистрацию
  • Добавить возможность удаления статусов
  • Заменить командную строку чем-то более простым и полезным. Либо упростить её и расширить её функционал
По всем вопросам о сотрудничестве с проектом пишите в Mastodon автора. Также вы можете ознакомиться с GitHub-репозиторием проекта.
